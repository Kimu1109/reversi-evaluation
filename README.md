# ReversiEvaluation

<img src="ReversiEvaluation-200x200.svg" alt="アイコン" />

リバーシの評価アプリケーションです。  
主に、大会や試合などで使われることを想定しています。

## 機能

#### 1. Edax-Reversiによる盤面評価

世界最強のオセロAIの1つであるEdax-Reversiによって、盤面を評価し、白側と黒側の勝率を計算し、表示します。  

#### 2. 戻す/進める

間違って打ってしまっても、盤面を戻したり、進めたりすることができます。

#### 3. スキップ

自由にターンをスキップすることができます。

#### 4. 勝率統計

勝率の推移をグラフにして表示することができます。  
ゲーム中に随時更新されます。

#### 5. ステータスバー

ウィンドウの下に、雑多な情報を表示します。  
勝率、ターン、スコア、ターンカウント、石の個数、ゲームは終了したか、などを表示します。

#### 6. 履歴の保存/読み込み

盤面の推移や評価値、勝率などをすべて保存し、いつでもゲームをやめて、再開することができます。  
保存時刻やプレイヤー名も保存されるので、大会や試合の盤面記録にも便利です。  

#### 7. Edax-Reversiの最善手の確認

Edax-Reversiの推奨手を保存し、確認することができます。  
あの時、どのように打っていれば、良かったのかなどを確認できます。
