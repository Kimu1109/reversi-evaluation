# ReversiEvaluation

<img src="ReversiEvaluation-200x200.svg" alt="アイコン" style="zoom: 33%;" />

リバーシの評価アプリケーションです。  
主に、大会や試合などで使われることを想定しています。

## 機能

#### 1. Edax-Reversiによる盤面評価

Edax-Reversiによって、盤面を評価し、白と黒の勝率を計算し、表示します。  
※Edax-Reversiが考える最善手を打ち続けた場合での勝率です。

#### 2. 戻す/進める

間違って打ってしまっても、盤面を戻したり、進めたりすることができます。

#### 3. スキップ

自由にターンをスキップすることができます。

#### 4. 勝率統計

勝率の推移をグラフにして表示することができます。  
ゲーム中に随時更新されます。

#### 5. ステータスバー

ウィンドウの下に、雑多な情報を表示します。  
勝率、ターン、スコア、ターンカウント、石の個数、ゲームは終了したか、などを表示します。
