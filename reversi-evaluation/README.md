# ReversiEvaluation 🎮📊

`ReversiEvaluation` は、C# と **Avalonia UI** で構築された、リバーシ（オセロ）の対戦およびリアルタイムAI解析を行うクロスプラットフォーム（Windows / Linux）対応のデスクトップアプリケーションです。

最強クラスのリバーシエンジンである **Edax** をバックグラウンドで動作させ、対局中の盤面評価値や勝率推移をリアルタイムに視覚化します。

---

## 🌟 主な特徴と魅力

### 1. 🤖 Edaxエンジンによる超高速・高精度なリアルタイム解析
世界最強クラスのリバーシ思考エンジン「**Edax**」をバックグラウンドプロセスとして起動。プレイヤーが石を置くたびに、Edaxが現在の盤面を分析し、**評価値（スコア）** と **最善手（おすすめの手）** を瞬時に提示します。

### 2. 📈 視覚的な勝率インジケーター & グラフ表示
- **勝率カラーバー**: Edaxの評価値をシグモイド関数に基づいて直感的な勝率（0%〜100%）に変換し、現在の優勢度をリアルタイムにカラーバーで表示します（「互角」「黒が優勢」「白がやや優勢」などのテキスト評価も連動）。
- **リアルタイム勝率グラフ**: **LiveChartsCore** を用いて、1手目から現在までの黒と白の勝率推移を滑らかな折れ線グラフで描画。対局の「流れ」や勝負の分岐点が一目で分かります。

### 3. 🔄 タイムトラベル（巻き戻し）＆ ブランチング機能
対局のすべての履歴が自動記録されています。
- 「戻す」「進める」ボタンを使って、過去の局面をいつでも確認できます。
- 過去の局面に戻って別の手を打つことで、**新たな対局ルート（ブランチ）へと分岐** させることが可能です（分岐以降の古い履歴は自動的に書き換えられます）。

### 4. 💻 モダンなデスクトップ体験
- クロスプラットフォーム対応の **Avalonia UI (v11.3.0)** と **Fluent Theme** を採用した、クリーンで使いやすいユーザーインターフェース。
- **CommunityToolkit.Mvvm** に基づくクリーンなMVVMアーキテクチャ。
- プレイヤー名のカスタマイズや、パス（スキップ）処理など、対局に必要な基本機能を完備。

## 📂 プロジェクト構造

```
reversi-evaluation/
├── Evaluations/
│   └── EdaxEvaluation.cs    # EdaxエンジンのI/O制御、評価値取得、勝率計算
├── Models/
│   └── ReversiGame.cs       # リバーシのゲームルール・盤面管理ロジック
├── ViewModels/
│   ├── ReversiViewModel.cs  # ゲーム状態の保持、Edax連携、グラフ描画データの制御
│   └── CellViewModel.cs     # 盤面の各マスの状態管理
├── Views/
│   ├── MainWindow.axaml     # メインウィンドウ
│   ├── ReversiBoard.axaml   # リバーシ盤面のカスタムコントロール
│   ├── PlayerInfo.axaml     # プレイヤー名と優勢度カラーバーの表示
│   ├── WinRateChart.axaml   # 勝率グラフ表示ウィンドウ
│   └── EvaluationHistoryWindow.axaml  # 履歴一覧表示ウィンドウ
└── edax/                    # OSごとのEdax実行バイナリ
```

## 🚀 クイックスタート

### 前提条件
- [.NET 10.0 SDK](https://dotnet.microsoft.com/download) がインストールされていること。

### 実行方法
1. プロジェクトのルートディレクトリで以下のコマンドを実行します。

```bash
dotnet run
```

*※ アプリ起動時に、OS（Windows / Linux）に適した `edax` バイナリが自動的に読み込まれます。*
